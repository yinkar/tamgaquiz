# Tamga Quiz · 𐱅𐰢𐰍𐰀

Göktürk (Orhon/Yenisey) alfabesini öğrenmek için React.js ile yapılmış quiz uygulaması.

Canlıya bak: https://tamgaquiz.netlify.app/

---

Bağımlılıkları kur: `npm install`

Geliştirme: `npm start`

Prodüksiyon: `npm run build`
